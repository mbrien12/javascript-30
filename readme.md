## # 3 - CSS variables
* You can create variables! Not using SCSS/SASS 
* Use ':root' e.g.
```
 :root {
      --base: #ffc600;
      --spacing: 10px;
      --blur: 10px;
    }

    img {
    padding: var(--spacing);
    background: var(--base);
    filter: blur(var(--blur));
    }

```
